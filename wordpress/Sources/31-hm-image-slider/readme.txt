A simple slider using in shortcode and widgets

== Description ==
This slider is a simple slider by using billboard jquery slider plugin with slider post type and some useful control for each slider.

== Installation ==
1. extract hm-image-slider folder to wp-content/plugins/
2. active plugin from plugin manager

= Using shortcode =
* create slider and use it\'s id in every post you want use
* use by this structure [hmslider id={$slider_id}]
